
# arduino_expander

Arduino as i2c (TWI) runtime configurable expander

Author: jkolczasty@gmail.com 

License: MIT

## Hardware

Tested on Orange Pi as I2C Master and I2C Slave:
  - Mini Pro (3.3V 8MHz)
  
## Implementation status

Implemented:
  - OUTPUT
  - INPUT
  - INPUT PULLUP
  - PWM
